# Lista de Compras a Proveedores Gratis

Aplicación web gratuita para crear **listas de compras a proveedores** con checklist interactivo de stock faltante.

## ✨ Características

- Checklist interactivo de artículos pendientes.
- Marca productos como comprados o elimínalos.
- Persistencia local (LocalStorage).
- Diseño Mobile-First optimizado.

## 🚀 Uso

1. Escribe el nombre del producto y cantidad.
2. Haz clic en **"Agregar"**.
3. Marca los productos a medida que los compras.
